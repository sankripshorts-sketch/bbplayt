# Эксплуатация (организационно)

- **Артефакты** билдов: хранение APK/IPA/ABB по политике компании; ссылки на тест-стенды.
- **Версии API:** при смене прокси BlackBears держите в синхроне `.env` и ветки приложения; тест `test:api` на стейдже.
- **Инциденты:** 403/500 массово — сначала **бэкенд/прокси**, не релиз клиента; клиент в основном stateless.
- **Мониторинг:** Sentry/аналитика не зашиты в этой доке по умолчанию — при подключении допишите раздел.
- **Пользователи:** FAQ и [troubleshooting.md](troubleshooting.md) как первый уровень поддержки.

См. также: [deployment.md](deployment.md), [api-and-backend.md](api-and-backend.md).
